# Movie Land

> Movie Land build with React Framework. App uses API for fetching movies.

## Built With

- React JS

## Getting Started

To get a local copy up and running follow these simple example steps.

- `git clone https://github.com/myaserkhan/Math-Magicians.git`
- `cd Math-Magician`
- `npm install`
- `npm start`

## Prerequisites

- Node.js
